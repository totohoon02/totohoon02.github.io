# SSE

- Server Sent Event
- 서버에서 클라이언트에 실시간으로 데이터를 전송하는 기술
- HTTP 연결을 길게 유지

```
 Client       Server
   | Connect    |
   | ---------> |
   |            |
   | Event      |
   | <--------- |
   |            |
   | Event      |
   | <--------- |
   |            |
   | Event      |
   | <--------- |
```

- 데이터의 변경이 있을 때 실시간으로 알려주기 위해 주로 사용
